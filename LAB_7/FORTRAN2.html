7. ОПЕРАТОРЫ УСЛОВИЯ

        В Фортране используются следующие логические операторы:
        .GT. (больше)
        .LT. (меньше)
        .EQ. (равно)
  1. Логический IF: IF (A .GT. B) MAX = A
  2. Блочный IF: IF (...) THEN ... ELSE ... ENDIF

8. СТРОКИ

        Для работы со строками используется тип CHARACTER.
        Доступны операции конкатенации (//) и выделения подстроки (:).

9. ФАЙЛОВЫЕ ОПЕРАЦИИ

        Основные команды для работы с файлами:
        OPEN - открытие файла.
        CLOSE - закрытие.
        REWIND - возврат в начало.
