ОБ ЭТОЙ ЧАСТИ

        Данный документ является кратким справочником по языку Фортран (Fortran).
        Фортран — первый язык программирования высокого уровня, получивший 
        широкое распространение. Предназначен в первую очередь для научных вычислений.

1. МЕТАКОМАНДЫ МС-ФОРТРАНА

        Метакоманды управляют процессом компиляции:
        $STORAGE:n - задает объем памяти для целых чисел.
        $INCLUDE:'filename' - включает текст из другого файла.
        $LIST - включает генерацию листинга.

2. ТИПЫ ДАННЫХ

2.1. Константы

        Константы в Фортране могут быть целыми (INTEGER), 
        вещественными (REAL), логическими (LOGICAL) или строковыми.

2.2. Переменные

        Имя переменной может содержать до 6 символов.
        Тип переменной определяется первой буквой (I-N для INTEGER по умолчанию).

3. СТРУКТУРА ПРОГРАММЫ

        Программа на Фортране состоит из программных единиц.
        Порядок строк:
        1. Заголовок (PROGRAM)
        2. Операторы описания типов
        3. Исполняемые операторы
        4. END
